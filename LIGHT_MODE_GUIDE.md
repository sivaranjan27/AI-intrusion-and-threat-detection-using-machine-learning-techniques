# How to Switch Between Light and Dark Mode

## Quick Instructions

The application now includes a **Theme Mode toggle** in the left sidebar!

### To Switch Modes:

1. **Look at the left sidebar** - You'll see a "Settings" section at the top
2. **Find "Theme Mode"** - It has two radio buttons: "Dark" and "Light"
3. **Click on "Light"** - The entire app will instantly switch to light mode
4. **Click on "Dark"** - Switch back to dark mode anytime

## What Changes?

### Dark Mode (Default)
- Background: Very dark blue/black (#0f1419)
- Text: Light gray (#e0e0e0)
- Cards: Dark with subtle borders
- Accents: Cyan (#00d4ff) and Purple (#7c5dff)

### Light Mode
- Background: White (#ffffff)
- Text: Dark (#1a1a1a)
- Cards: Light gray (#f8f9fa)
- Accents: Purple (#6b4ee8) and Blue (#0095cc)
- Better contrast for readability

## Features Work the Same in Both Modes

✓ Login & Attack Detection - Tab 1
✓ Phishing URL Analysis - Tab 2  
✓ DOS Information - Tab 3
✓ Chatbot - In the sidebar
✓ All colors automatically adjust

## Troubleshooting

If text is still hard to read:

**For Light Mode:**
- Make sure you selected "Light" radio button
- Try refreshing the page (F5 or Cmd+R)
- Check if your browser has a dark mode extension enabled

**For Dark Mode:**
- Make sure you selected "Dark" radio button
- The app should show light text on dark background
- All UI elements have good contrast
